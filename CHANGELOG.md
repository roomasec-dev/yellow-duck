# 更新日志

## [1.0.0] - 2026-05-15

### Added
- 初始版本发布
- 基于 Go 的 AI 控制面脚手架能力，覆盖配置、路由、会话、调度与工具编排
- 多渠道接入能力：飞书、钉钉、企业微信、Slack，支持机器人消息收发与统一协议层处理
- EDR 能力集成：
	- 主机管理（查询、隔离、移除）
	- 事件与风险检出查询（列表、详情、分页检索）
	- 指令下发与任务结果查询
	- 策略、IOA、IOC、病毒相关能力与批量处理接口
    - 威胁狩猎日志调查
- 高危写操作确认流程，降低误操作风险
- 模型分层能力：主模型、路由模型、进度播报模型协同，支持长耗时任务过程反馈
- 提示词与技能体系：
	- 内置 AGENTS 提示词与大量 EDR/调度技能模版
	- 支持技能化调用与能力扩展
- 超大详情处理链路：artifact 存储 + 分段读取/搜索，避免一次性注入超大上下文
- 知识库能力：支持自然语言检索与增删改维护，支持外部目录挂载
- 长期记忆能力：支持会话记忆持久化与上限裁剪
- 数据持久化：基于 SQLite（无 CGO）实现会话、摘要与消息去重落库

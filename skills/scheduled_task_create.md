# Tool: scheduled_task_create

用途：
- 创建一个定时任务，让系统按固定频率自动执行某段安全运维任务。

适用场景：
- “你五分钟执行一次，帮我看看有没有最新告警”
- “一小时一次，帮我 hunt 一下内网有没有恶意文件”

输入建议：
- `task_prompt` 写清楚真正要执行的任务内容。
- `task_interval_minutes` 如果用户没明确说时间，默认填 `5`。
- `task_title` 可简短概括任务用途；没有也可以让系统自动生成。

规则：
- 定时任务默认是持续执行，不需要每次再问用户确认。
- 定时任务执行时由子agent自己干活，不要让它每次回到“你要我查什么”的提问模式。
- 新任务应绑定当前会话，后续结果回到当前聊天里。

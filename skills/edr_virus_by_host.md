# Tool: edr_virus_by_host

用途：
- 按主机查询病毒信息，适合回答"某台主机上有哪些病毒""某台主机是否中毒"。

输入建议：
- 最好带 hostname 或 client_id 来精确定位主机。
- 支持分页查询，page 和 page_size 一起规划。
- 如果用户提到第几页、下一页、每页多少条，要把 page 和 page_size 一起规划出来。

输出建议：
- 优先展示主机名、病毒名称、病毒类型、发现时间、威胁等级。
- 如果结果为空，明确说"该主机未发现病毒"或"当前没有匹配的病毒记录"。
- 按最新发现时间优先理解结果。
- 回复里要保留当前页、总页数、是否还有下一页，方便继续翻页。

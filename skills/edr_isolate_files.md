# Tool: edr_isolate_files

用途：
- 查询被隔离的文件列表，适合回答"有哪些文件被隔离了""某台主机隔离了哪些文件""隔离文件的处置状态"。

输入建议：
- 支持分页查询，page 和 page_size 一起规划。
- 如果用户提到第几页、下一页、每页多少条，要把 page 和 page_size 一起规划出来。
- 如果用户只说"下一页"，优先根据最近一次隔离文件列表的页码继续翻页。

输出建议：
- 优先展示文件 GUID、主机名、文件名、哈希值（MD5/SHA1）、状态（隔离中/已申请恢复/已恢复）。
- 按最新隔离时间优先理解结果。
- 重点关注状态为"隔离中"的文件，这些可能需要进一步处置。
- 回复里要保留当前页、总页数、是否还有下一页，方便继续翻页。
- 如果结果为空，明确说"当前没有查到隔离文件"。

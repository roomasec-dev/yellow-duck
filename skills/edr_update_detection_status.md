# Tool: edr_update_detection_status

用途：
- 更新检测记录的状态（标记为已处理、误报等），属于高危写操作。

风险要求：
- 必须等待用户明确回复"确认"后再执行。
- 执行前要确认检测 ID 和处置状态明确无歧义。

输入说明：
- detection_id：检测 ID，必填。
- status：处置状态（整数），必填。

状态映射表：

| 用户说法 | status |
|---------|--------|
| 待处置 | 1 |
| 处置中 | 2 |
| 已处置、已处理、标记为已处理 | 3 |
| 误报、标记为误报、false positive | 4 |

输出建议：
- 成功后汇报更新的检测数量。

# Tool: edr_virus_hash_hosts

用途：
- 按哈希查询关联主机，适合回答"某病毒哈希在哪些主机上出现过""某哈希的主机分布"。

输入建议：
- 最好带 hostname 或 client_id 来缩小范围。
- 支持分页查询，page 和 page_size 一起规划。
- 如果用户提到第几页、下一页、每页多少条，要把 page 和 page_size 一起规划出来。

输出建议：
- 优先展示主机名、病毒名称、首次发现时间、最后发现时间、发现次数。
- 如果结果为空，明确说"该哈希未在任何主机上发现"。
- 按最后发现时间优先理解结果。
- 回复里要保留当前页、总页数、是否还有下一页，方便继续翻页。

# Tool: edr_strategies

用途：
- 查询策略列表，适合回答"有哪些策略""策略的配置情况"。

输入说明：
- instruction_name：策略类型，必填，如"virus_scan_settings"。

输入建议：
- 支持分页查询，page 和 page_size 一起规划。
- 如果用户提到第几页、下一页、每页多少条，要把 page 和 page_size 一起规划出来。

输出建议：
- 优先展示策略 ID、名称、类型、状态（1=启用，0=禁用）、范围类型。
- 按最新更新时间优先理解结果。
- 回复里要保留当前页、总页数、是否还有下一页，方便继续翻页。
- 如果结果为空，明确说"当前没有查到策略"。

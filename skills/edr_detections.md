# Tool: edr_detections

用途：
- 查询近期行为检出，适合回答“最近有哪些高风险动作”“最近有没有异常行为命中”。

输入建议：
- 这个工具偏全局态势，适合没有明确主机时使用。
- 如果用户提到第几页、page、下一页、每页多少条，要把 page 和 page_size 一起规划出来。
- 如果用户只说“下一页”，优先根据最近一次 detections 列表的页码继续翻页。

输出建议：
- 默认按最新时间优先理解结果，先看最近发生的 detection。
- 重点看 threat_level、host_name、deal_status、root_name。
- 如果用户在排查风险，优先把高等级条目放前面。
- 回复里要保留当前页、总页数、是否还有下一页，方便继续翻页。
